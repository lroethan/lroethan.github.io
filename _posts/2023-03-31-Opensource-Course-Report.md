Course Report on General Studies in the Open Source Community.

Blog overview
======

I believe the original purpose of blogging is to *record* and *share* information. Recording involves summarizing one's own learning process and problems encountered, while sharing is about providing guidance to others with easy-to-understand content. However, I am not accustomed to sharing content on my blog, and instead prefer to use my own personal knowledge management system like a *Notion* or *Yuque* to record my thoughts. Therefore, this blog will only periodically record certain content. Without further ado, this blog consists of four parts:

1. About Me
2. Publications
3. Teaching
4. Posts

This blog is built using the simplest static site generation method. It only requires updates to Markdown files and some configuration scripts to implement. Currently, this blog has not been officially launched, and I hope to update it with some personal essays in a timely manner in the future.

Overall, at the moment, I have only left some personal information on this blog and have reserved many areas to explore. Please look forward to the future of my blog.
